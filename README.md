
# React Custom Hooks Starter (JS)
실습용 Vite 프로젝트다.

## 요구사항
- Node.js 18+

## 설치
```bash
npm i
npm i -D @vitejs/plugin-react
npm run dev
```

## 포함된 훅
- useToggle, usePrevious, useLocalStorage, useDebounce
- useEventListener, useInterval, useAsync, useFetch, useOnClickOutside

## 구조
```
src/
  hooks/
  examples/
  App.jsx
  main.jsx
```
